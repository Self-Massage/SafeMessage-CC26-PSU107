# Data Science - SafeMessage

Folder ini berisi seluruh pekerjaan tim Data Science untuk proyek SafeMessage, 
sistem deteksi phishing message Bahasa Indonesia berbasis NLP menggunakan TextCNN.

## Struktur Folder

| Folder | Deskripsi |
|--------|-----------|
| `notebooks/` | Jupyter Notebook untuk data wrangling dan EDA |
| `dashboard/` | Kode Streamlit untuk visualisasi dan monitoring model |
| `ab_testing/` | Implementasi A/B Testing antar konfigurasi model |
| `data/` | Dataset yang digunakan (tidak di-push ke repo) |
